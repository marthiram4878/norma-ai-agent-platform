## 🗂️ How it Works

Norma uses several technologies to complete tasks for you:

*   RAG (Retrieval-Augmented Generation): The software searches your specific files to find answers instead of guessing.
*   Long-term Memory: Norma records previous interactions to learn your specific requirements over time.
*   Agentic Workflows: The system breaks large goals into smaller, manageable steps, executes them, and reports back to you.

## 🛡️ Privacy and Data

Norma values your privacy. The system processes your information locally on your computer whenever possible. Your document content stays on your hard drive unless you explicitly choose to share it with an external service provider for specific advanced task processing. You control every data source added to the system through the Settings menu.

## 🔧 Troubleshooting

If you encounter issues, check these common items:

*   Application does not open: Ensure you have the latest updates for Windows. Restart your computer.
*   Agent fails to read files: Verify the files are in a supported format like PDF, TXT, or DOCX. Ensure the agent has permission to access the selected folder in Windows settings.
*   Slow performance: Close other high-demand applications like video editors or browsers with many open tabs.
*   Error messages: Copy the error text and look for it in the settings help menu.

## 📈 Updating the Software

Norma checks for updates automatically each time you open the program. If an update exists, a notification appears in the top corner of the screen. Click the Update button to install the latest improvements and security patches. You do not need to uninstall the old version; the installer handles everything for you.

## 📄 Managing Your Knowledge Base

Your knowledge base serves as the brain for your agents. To keep things organized:

1. Organize documents into folders based on topics or projects.
2. Remove old folders that you no longer need via the Sources tab to improve search speed.
3. Use clear file names. The system interprets file names to provide better context to the agents.

## 🎯 Best Practices

*   Keep your instructions brief and specific.
*   Provide clear examples if you need the agent to follow a specific format.
*   Review the output of your agents regularly to ensure accuracy.
*   Assign primary agents for specific workflows to keep your project environment clean.
